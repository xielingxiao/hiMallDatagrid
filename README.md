# hiMallDatagrid
基于jquery的表格初始化插件，适用于后台表格数据输出等

hiMallDatagrid插件主要用于后台繁琐的表格数据初始化，异步翻页，条件筛选、按字段排序等等功能；


详细参数配置请参见插件底部defaults对象，方法参见插件对外公开的所有方法集合；
